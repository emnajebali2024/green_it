# Green IT Challenge

Application web éco-conçue pour le défi Green IT organisé par NoctEvent et O2Switch.

## Caractéristiques techniques
- Poids total : < 10 KB
- Éléments DOM : < 50
- Requêtes HTTP : 3
- Score EcoIndex : > 85/100
- Compatible PWA

## Installation
1. Déployer les fichiers sur un serveur web
2. Vérifier que PHP est installé pour l'API
3. Accéder à index.html

## Tests
- Google Lighthouse
- EcoIndex.fr
- Website Carbon Calculator
